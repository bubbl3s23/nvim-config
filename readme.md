# Neovim Configuration

Simple Neovim configuration built for the **nightly snapshot** and the native Pluign Manager.
Written in an evening with enough features for a comfortable dev experience, might grow in complexity later.

## Features

- **Plugin Management**: Uses built-in `vim.pack.add()` (no plugin manager required)
- **LSP**: nvim-lsp
- **Completion**: Blink.cmp with LuaSnip snippets
- **Navigation**: FZF-based file/buffer/grep searching with `fzf-lua`
- **Quality of Life**: Smooth scrolling, which-key hints, todo comments, and more
